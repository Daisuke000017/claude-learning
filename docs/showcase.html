WORKFLOW

4ステップの作業フロー

人間がやったのは「指示を出すこと」だけ。残りはすべてClaudeが自動で実行しました。

1

Webページの分析・コンテンツ抽出

Anthropic Academyの各コースページにアクセスし、レッスン内容を自動で読み取り・分析しました。

  • ブラウザを自動操作してコースページを順番にアクセス
  • 各レッスンの学習目標・キーポイント・演習内容を抽出
  • 13レッスン分のノートを自動生成（約920行）
  • コース構成を分析し、セクション構造を設計
2

PowerPoint資料の自動作成

分析したコンテンツをもとに、デザイン統一されたスライド資料をプログラムで自動生成しました。

  • PptxGenJSライブラリを使ったNode.jsスクリプトを作成
  • 統一カラーパレット（Ocean Gradient）でデザイン
  • タイトル・目次・解説・まとめを含む構成
  • 3コース合計70枚以上のスライドを生成
3

Webページの構築

スマホ対応のレスポンシブWebページを、HTML/CSS/JSで一から作成しました。

  • フェードインアニメーション付きのモダンなデザイン
  • PC・タブレット・スマホすべてに対応（レスポンシブ）
  • Lucide Iconsを使ったアイコン表示（絵文字不使用）
  • 目次ページ含む計4つのHTMLファイルを生成
4

GitHubへのアップロード・公開

作成したファイルをGitHubリポジトリにプッシュし、GitHub Pagesでサイトとして公開しました。

  • Gitリポジトリの初期化とファイルのコミット
  • GitHubへの自動プッシュ
  • GitHub Pages用のディレクトリ構成に調整
  • 公開URLでアクセス可能に
